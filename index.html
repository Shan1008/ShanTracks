Sync Settings
1. Go to github.com → Settings → Developer settings → Personal access tokens → Tokens (classic)
2. Click Generate new token (classic). Give it a name like "ShanTracks".
3. Under scopes, check only gist. Set expiry to No expiration.
4. Copy the token (starts with ghp_) and paste above.
5. Go to gist.github.com → Create a new secret gist. Filename: shantracks_init.json, content: {}. Multiple users can share one Gist — each gets their own file automatically.
6. Copy the Gist ID from the URL (the hash after your username) and paste above.
7. Click Save & Connect. Your data syncs to shantracks_YourUserID.json inside the Gist.
